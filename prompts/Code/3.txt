Тебе дан 0-индексированный массив целых чисел nums, содержащий n различных положительных целых чисел. Перестановка массива nums называется особой (special), если выполнено условие:
Для всех индексов 0 <= i < n - 1 выполняется либо nums[i] % nums[i+1] == 0, либо nums[i+1] % nums[i] == 0.
Нужно вернуть общее количество особых перестановок. Так как ответ может быть очень большим, верни его по модулю 10^9 + 7.

Пример 1:
Ввод: nums = [2,3,6]
Вывод: 2
Пояснение: [3,6,2] и [2,6,3] — две особые перестановки массива nums.

Пример 2:
Ввод: nums = [1,4,3]
Вывод: 2
Пояснение: [3,1,4] и [4,1,3] — две особые перестановки массива nums.

Ограничения:
* 2 <= nums.length <= 14
* 1 <= nums[i] <= 10^9
